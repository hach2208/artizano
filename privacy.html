7. Sécurité des données

Nous mettons en œuvre les mesures de sécurité suivantes :

  • Chiffrement des communications via HTTPS/TLS 1.3
  • Mots de passe stockés avec hachage bcrypt (jamais en clair)
  • Row Level Security (RLS) PostgreSQL sur toutes les tables
  • Authentification JWT avec expiration courte (1 heure)
  • Clés API sensibles stockées dans des coffres-forts sécurisés
  • Sauvegardes automatiques quotidiennes chiffrées
  • Accès aux données de production limité aux administrateurs
  • Journaux d'audit pour les opérations sensibles

⚠️ En cas de violation de données, vous serez notifié dans les meilleurs délais et la CNIL sera informée sous 72 heures, conformément à l'Art. 33 RGPD.
