⚙️ Configuration

1

Clé Google Places API

Va sur console.cloud.google.com → Active Places API → Crée une clé API.

GOOGLE_PLACES_API_KEY=AIzaSy-xxxxxxxxxxxxx
2

Clé Anthropic (Claude AI)

Va sur console.anthropic.com → API Keys → Crée une clé.

ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx
3

Config SMTP (envoi d'emails)

Pour Gmail : active Mots de passe d'application dans ton compte Google.

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=ton@gmail.com
SMTP_PASS=xxxx xxxx xxxx xxxx
4

Lancer le serveur

Copie .env.example en .env, remplis tes clés, puis :

cd mapleads && npm install && npm start
